### 메타데이터 플러그인

Plex, KODI, Jellyfin 등과 연동하여 메타데이터를 제공하는 플러그인이다.


첫 release 후 3년이 지나 현재 구조에 맞지 않는 오래된 코드도 같이 있다.


[한시오분](https://github.com/105PM)님이 특정 메타를 전담하여 개발중이다.
